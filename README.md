## Introduction

*ninja* is a distributed build system of [distninja](https://github.com/distninja) written in Rust.



## Prerequisites

- Rust >= 1.60.0



## Run

```bash
make install
make build
./target/release/ninja
```



## Docker

```bash
make docker
docker run ghcr.io/distninja/ninja:latest
```
